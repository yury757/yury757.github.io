六、拦截器和过滤器

拦截器：拦截器只会拦截访问的控制器方法，如果访问的是jsp、html、css、image、js是不会被拦截的。实现了HandlerInterceptor接口的类就是拦截器。拦截器是AOP思想的一个具体应用。

过滤器：在web.xml中配置的Filter就是过滤器，url_pattern配置了/*会对所有资源进行过滤。
